# Lab 1

Lab experiment based on the following stack:
- Java 17
- Spring Boot 2.7.x
    - Spring Data REST + JPA

## How to...

|                                               | Linux / macOS            | Windows                    |
|-----------------------------------------------|--------------------------|----------------------------|
| Run the application <br>http://localhost:8080 | `./mvnw spring-boot:run` | `mvnw.cmd spring-boot:run` |
| Run the test suite                            | `./mvnw test`            | `mvnw.cmd test`            |
